# Sismos en Perú

Aplicación web para explorar y visualizar los sismos registrados en el Perú a lo largo de los años.

## Integrantes
- Jhosselyn Dayanna Enriquez Aliaga
- Jose Luis Cepida Castellares

## ¿Qué puedes hacer en la app?
- Ver todos los sismos registrados en un gráfico y mapa
- Filtrar por magnitud, profundidad, década y escala de Richter y profundidad a la vez.
